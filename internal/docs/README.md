# Internal Maintainer Docs

Documents in this directory are internal ShakaCode maintainer material for the
`react-on-rails-rsc` package. They may describe unpublished release processes,
agent workflows, implementation tradeoffs, temporary diagnostics, or downstream
test tactics.

Do not treat this directory as the public documentation surface. Public docs for
React on Rails application developers belong in downstream `shakacode/react_on_rails`
documentation.
