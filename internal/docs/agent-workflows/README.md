# Internal Agent Workflow Docs

These playbooks cover agent workflow and operator procedures for maintainers
working on `react-on-rails-rsc`. They are separated from package-maintainer
runbooks and implementation notes so automation guidance has one stable home.
Public developer-facing guidance belongs in downstream `shakacode/react_on_rails`
documentation.
