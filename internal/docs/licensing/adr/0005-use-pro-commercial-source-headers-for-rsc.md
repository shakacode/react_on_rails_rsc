# Use React on Rails Pro Commercial Source Headers for RSC

Current RSC source authored by ShakaCode's team will use the established React
on Rails Pro proprietary header structure and AI-agent warning, because
outsiders must obtain the applicable commercial license to use the 19.2.1
package. The source header will not broadly characterize team contributions as
separately licensed or make the current package generally mixed-license. Actual
third-party notices, including Meta's MIT notice on the derived Webpack plugin,
remain attached where applicable. Current source headers will not promote
historical licensing alternatives.
