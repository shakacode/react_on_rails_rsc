# Keep the RSC Repository and Focus Current License Messaging

The existing `react_on_rails_rsc` repository and history will remain in place
rather than being deleted and republished. Current headers, README, changelog,
and package metadata will clearly state that version 19.2.1 onward is
commercially licensed without repeatedly advertising or encouraging use of
older MIT releases. Applicable third-party notices, including Meta's MIT notice,
remain preserved where required.
