# Version the Corrected Canonical EULA as 2.3

The corrected canonical EULA will be version 2.3, dated July 14, 2026, and will
apply prospectively to `react-on-rails-rsc` 19.2.1 and React on Rails Pro
artifacts published with the next 17.0.0 release candidate. The version change
makes the revised grant, attribution obligation, and authorized-free-use
language auditable without using current product messaging to promote older
RSC releases.
